望月 英樹

題名:
Web サービスを利用した関連文生成に関する研究
‐Google API を例に‐

序論

情報化社会の現在，多種多様な情報が散在している．存在する情報源から，自分で情報を獲得しに行かなければならないため，目的の情報の獲得以外に手が回らなくなることは，十分に考えられる．目的の情報以外の関連情報や有用になりうる情報を，効率よく自動的な提供が必要となる．

目的

日常何か文章を書くことで，様々な情報を取り入れ，いつも単調な文章を書いていたとしても，いろいろな分野の関連情報を入手し，発想の拡大や知識の向上が期待できるシステムの構築である．文章を人に見せるだけでなく，自分に魅（み）せるために書く価値が，生まれるのである．

if 文の構文　（参考書第 4 章 pp. 75- pp. 101）

if ( 条件 ) {

　　 条件が成り立つときに実行される式 ;

}

括弧は必ずつけておこう （参考書 pp. 144）

if ( score >= 60 )

　　printf("60点以上です");

　　printf("あなたは合格です");

if と else の組み合わせ　（課題　03A1）

if ( score >= 60) {
    printf("60点以上です");
} else {
    printf("また来年会いましょう");
}

if 文の代わりになる式 (1)　（課題 03A2）

良くある一句:

if (m > n)
    r = m;
else
    r = n;

特に r がもう少し複雑な場合には次が有効:

r = (m > n) ? m : n;

if 文の代わりになる式 (2)　（課題 03A2）

|| や && で if 文のような物が作れる:

if ( score >= 60) {
    printf("60点以上です");
} else {
    printf("また来年会いましょう");
}

の代わりに

score >= 60 && printf("60点以上です") || printf("また来年会いましょう");

switch 文（課題 03C2）参考書 5 章 pp. 106-111

switch ( 式 ) {
  case 定数1 :
    処理1 ;
    break;
  case 定数2 : case 定数3:
    処理2 ;
    ...
    break;
  default :
    処理3 ;
    break;
}
